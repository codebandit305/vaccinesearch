Welcome!

This website helps you locate pharmacies in New York State that provide COVID-19 vaccines. Once you select a pharmacy location, you can see the type of vaccine available and the availability of appointments. You will not be able to make a vaccination appointment directly from this website. However, you can book an appointment via the provided link to the pharmacy site once you make a selection.
