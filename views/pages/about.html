About the Website

I created this website originally as a fun project for myself. I did not think beyond showing locations on a map. However, as I added new features, I wanted the website to become a useful tool for finding COVID-19 vaccines. I am still learning and improving the application, so if there are bugs or you have suggestions on how to improve the website, feel free to contact me at vaccinesnearyou@gmail.com.

How I Did It

I use HTML, CSS, and Javascript to display everything you see (map, text, images, etc). I host the server on Digital Ocean and use Node.js and NGINX for the backend (routing, updating files, etc). The data on vaccine providers is collected from an API created by Vaccine Spotter.
